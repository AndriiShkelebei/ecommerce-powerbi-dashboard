# SQL Scripts

This folder contains SQL queries used for data preparation and analysis.

## Files

- `01_create_analysis_table.sql` – builds the main analytical dataset
- `02_business_analysis_queries.sql` – business analysis queries (KPIs, trends, retention)
- `03_retention_analysis.sql` – retention and customer behavior analysis (customer segmentation, repeat purchases, category-level retention, revenue contribution, delivery impact, data validation)
